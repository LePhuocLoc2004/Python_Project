# 🧾 Billing System using Python Tkinter & SQLite

A simple desktop **Billing System application** built with **Python, Tkinter GUI, and SQLite database**.  
This application allows users to manage products, create bills, and automatically update stock.

---

# 🚀 Features

- 📦 Product Management
  - Add new products
  - Store product ID, name, price, and stock
  - View product list

- 🛒 Billing System
  - Add items to bill using Product ID
  - Specify quantity
  - Automatic subtotal calculation

- 📊 Stock Management
  - Check product availability
  - Prevent selling out-of-stock items
  - Update stock after generating invoice

- 🧾 Invoice Generation
  - Generate invoice with total amount
  - Save billing data to SQLite database

---

# 🛠 Technologies Used

- **Python**
- **Tkinter** (GUI framework)
- **SQLite3** (Database)
- **VS Code** (Development environment)
